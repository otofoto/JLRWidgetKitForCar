## ✨ 功能特色

### 🚗 全方位车辆监控
- **基础信息**
  - 实时油量和续航里程
  - 总里程数和保养提醒
  - 车辆位置信息
  
- **安全状态**
  - 车门锁定状态
  - 车窗开关状态
  - 引擎盖/后备箱状态
  - 四轮胎压监测
  
### 🎨 精美视觉体验
- **丰富的小组件尺寸**
  - 锁屏小组件（圆形/行内/矩形）
  - 主屏幕小组件（小号/中号/大号）
  
- **个性化界面**
  - 支持自定义背景图片
  - 可调节背景暗度
  - 智能状态配色
  - 优雅的信息布局

### ⚡️ 智能化体验
- **灵活的数据更新**
  - 自定义更新周期
  - 智能缓存机制
  - 支持手动刷新
  - 显示双重时间戳
  
- **多车辆支持**
  - 支持多车辆管理
  - 快速切换监控车辆
  - 独立配置保存

## 🚀 快速开始

### 系统要求
- iOS 16.0 或更高版本
- Scriptable App
- 路虎 InControl 账号

### 安装步骤
1. 在 App Store 下载并安装 [Scriptable](https://apps.apple.com/app/scriptable/id1405459188)
2. 下载本项目的最新版本脚本
3. 将脚本导入到 Scriptable
4. 运行脚本并按提示配置
5. 在主屏幕或锁屏添加小组件

### 使用说明
1. **首次配置**
   - 登录路虎账号
   - 选择要监控的车辆
   - 设置更新周期
   
2. **添加小组件**
   - 长按主屏幕或锁屏
   - 点击添加小组件
   - 选择 Scriptable
   - 选择本脚本
   
3. **个性化设置**
   - 自定义背景图片
   - 调节背景暗度
   - 设置更新频率

## 🛠 高级配置

### 更新周期设置
- 最短支持 30 秒
- 推荐设置 10 分钟以上
- 可根据实际需求调整

### 背景图片设置
- 支持本地图片
- 支持 iCloud 图片
- 可调节暗度 0-100%

### 数据缓存管理
- 自动缓存机制
- 支持手动清除
- 智能过期处理
